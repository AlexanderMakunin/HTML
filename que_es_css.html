¿Qué es CSS y para qué sirve?

CSS, o Cascading Style Sheets, es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML. Su función principal es permitir a los desarrolladores web separar el contenido (HTML) de su presentación visual (colores, fuentes, espacios, etc.). Esto facilita el diseño y la modificación de páginas web, ya que permite aplicar estilos de manera consistente en múltiples elementos y páginas, mejorando la mantenibilidad y el control del diseño.

    • CSS en línea (inline) Este método consiste en aplicar estilos directamente a un elemento HTML mediante el atributo style. Aunque es fácil de usar, no es recomendable para grandes proyectos porque hace que el código sea más difícil de mantener.
    <"p style="color: blue; font-size: 20px;">Este es un párrafo con estilo en línea.
