Conceptos Básicos de CSS

CSS Margin

El margin (margen) se utiliza para crear espacio externo alrededor de un elemento. Se puede aplicar a todos los lados y se puede ajustar de forma individual. El margen no afecta el tamaño del elemento, sino que se añade fuera de su borde.

CSS Padding

El padding (relleno) se utiliza para crear espacio interno dentro de un elemento, entre su contenido y su borde. Al igual que el margen, el relleno puede ser ajustado en cada lado y afecta el tamaño total del elemento.

CSS Border

El border (borde) es una línea que rodea el contenido de un elemento. Se puede personalizar en grosor, estilo y color. El borde se encuentra entre el margen y el relleno, y su tamaño afecta el tamaño total del elemento.

CSS Width and Height

Las propiedades width (ancho) y height (alto) determinan el tamaño de un elemento. Se pueden establecer en valores fijos o porcentuales. El tamaño total del elemento también se ve afectado por el padding y el border.
